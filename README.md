## ¿Por qué Morat salvo mi vida?

### Contexto 

+ **Estado mental cuestionable**

### ¿Quién es Morat?

![Morat](./images/morat.png)

+ Cuatro colombianos que tocan pop-folk latino

+ Especialistas en canciones que suenan felices pero te destruyen con educación.

+ Soundtrack oficial de rupturas que _ni siquiera fueron relaciones_

### ¿Qué significa que Morat **salvo mi vida**?

+ Hicieron cantables mis problemas

+ Y ya, con eso basta

### La formula de Morat

+ Letras simples pero que pegan

+ Ritmos alegres para llorar con dignidad

### Impacto emocional

+ La tipica frase: _"Se lo conte en confianza y escribió una canción"_

+ Sensación de no soy la unica pasando por esto 

### Canciones que me marcaron 

+ Valen más -> _mensaje profundo (para mi)_

+ ¿A dónde vamos? -> _canción que tengo con mi novio_

+ Antes de los veinte -> _A quien no?_

### Psicología pop

+ LLorar cantando 

+ Letras suficientemente ambiguas para que apliquen a todo

+ Memoria emocional: Canciones = cápsulas del tiempo.

## Morat vs. terapia tradicional

| Morat | Terapia |
|------|---------|
| Gratis (con Spotify) | Costosa |
| Disponible 24/7 | Cita previa |
| No hace preguntas | Hace demasiadas |
| Te entiende sin explicarte | Te pide que expliques todo |
| Ideal para crisis a las 2 a.m. | Horarios laborales |
| No soluciona nada | Tampoco, pero ayuda |

> *Nota: esto es humor. La terapia sí es importante.*

### Riesgos

+ Pensar: “esta canción me entiende”

+ Reproducir la misma canción 47 veces.

+ Idealizar el pasado

### Conclusión

Morat no salvó mi vida de forma literal. La hizo:

+ más soportable,

+ más narrable,

+ y con mejor soundtrack
